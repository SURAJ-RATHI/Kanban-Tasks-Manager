
# Kanban Todo  App

A simple full-stack Kanban app with:
- `frontend`: React + Vite UI
- `backend`: Node.js + Express API

Watch the demo here:

## Project Structure

```text
Kanban App/
  backend/
  frontend/
```

## Setup

```bash
git clone https://github.com/SURAJ-RATHI/Kanban-Task-Manager.git
cd Kanban-Task-Manager
```

## Run 

Open **two terminals** in the project root (`Kanban-Task-Manager`) and run:

### Terminal 1 (Backend)

```bash
cd backend;npm i; npm run dev
```

### Terminal 2 (Frontend)

```bash
cd frontend; npm i; npm run dev
```

## Default URLs

- Frontend: `http://localhost:5173`
- Backend: `http://localhost:5000`
